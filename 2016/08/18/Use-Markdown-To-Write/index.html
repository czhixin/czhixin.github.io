迷上用Markdown写作

发表于 2016-08-18

Markdown让我们专注写作，而不是关注排版，因为你不需要花心思去排版。

现在排版工具足够多选择，不少编辑器在往复杂界面的路上越走越远，不断考验使用者的智商。但是即便是功能强大且复杂的编辑器仍有大部分人排不出满意效果，极少工具考虑排版的简易使用和美学相结合。

我看到Markdown，第一反应是这货不就和Wiki编写语法相似吗。还得学习语法？！想着就放一边了，后来咬牙一看，不到5分钟看完，语法就那么简单。再试试去写一份文档，噢，发现无可救药的爱上用Markdown写作，于是拿之前的几篇文章排版一遍，整个过程不到5分钟全排完。

当然也是由于我之前的排版不复杂，但是一下子就改掉用其他编辑工具排出来灰头土脸的感觉，美感凸显，不得不说——我迷上用Markdown写作。于是后面写文章、写技术文档，甚至写方案都用上。

简单介绍一下。Markdown是一种可以使用普通文本编辑器编写的标记语言,说得有点复杂了，其实就是来排版文章的，你可以借助各种工具MarkdownPad、Mou、Atom等工具进行书写与排版，这些工具跟记事本很接近，就比记事本复杂一点。

最后，说说不好的地方吧。

  1. 依赖Markdown环境（可以通过生成解决）
  2. 图片排版麻烦
  3. 其它
